# Trabajo Integrador - Algoritmos y Complejidad

## 📌 Descripción
Proyecto integrador de la cátedra **Algoritmos y Complejidad**, orientado al desarrollo de soluciones con **Python** y **Google Colab**.  
Incluye prácticas de optimización, manejo de datasets y aplicación de técnicas de **Inteligencia Artificial**.

## 🎯 Objetivos
- Aplicar conceptos de programación y análisis computacional.
- Implementar algoritmos de optimización en problemas prácticos.
- Utilizar datasets para entrenar y evaluar modelos.
- Documentar el proceso de desarrollo con claridad y coherencia.

## 🛠️ Tecnologías utilizadas
- **Python 3**
- **Google Colab**
- **NumPy / Pandas**
- **Matplotlib / Seaborn**
- **Scikit-learn**

## 📂 Estructura del repositorio
- `INTEGRADOR_AyC.ipynb` → Notebook principal con el desarrollo del proyecto.
- `datasets/` → Conjunto de datos utilizados.
- `docs/` → Documentación y recursos adicionales.
- `README.md` → Descripción general del proyecto.

## 🚀 Ejecución
1. Abrir el notebook en Google Colab.
2. Cargar los datasets desde la carpeta correspondiente.
3. Ejecutar las celdas en orden para reproducir los resultados.

## 📊 Resultados esperados
- Visualizaciones de datos y métricas de evaluación.
- Implementación de modelos de optimización.
- Conclusiones sobre la aplicación de técnicas de IA en problemas reales.
